Test de normalidad

Para determinar si nuestros datos siguen una distribución normal, se aplican pruebas de normalidad que comparan la distribución empírica con una distribución normal teórica; gráficamente, la normalidad se refleja en una curva de campana simétrica centrada en la media. Entre las pruebas más utilizadas se encuentran Shapiro-Wilk, Kolmogorov-Smirnov y Anderson-Darling, cada una con sensibilidad distinta según el tamaño de muestra y la forma de los datos. Estas pruebas evalúan si la diferencia entre ambas distribuciones es lo suficientemente pequeña como para aceptar la hipótesis de normalidad, lo cual se interpreta mediante el valor p: si el p-value es menor a 0.05, se rechaza la hipótesis nula y se concluye que los datos son estadísticamente significativos; si es mayor, no hay evidencia suficiente para rechazarla. En nuestros casos de estudio, se aplicaron principalmente las pruebas de Shapiro-Wilk y Wilcoxon, esta última útil para comparar medianas en muestras relacionadas cuando no se cumple la normalidad. La elección entre ambas depende del tipo de variable, el tamaño muestral y la distribución observada, y permite establecer si los datos presentan diferencias significativas o si se mantienen dentro del rango esperado bajo la hipótesis nula.

Pruebas paramétricas y no paramétricas

Pruebas paramétricas

Las metodologías paramétricas suponen que los datos provienen de una población con distribución normal. Estas pruebas suelen tener mayor poder estadístico y son ampliamente utilizadas en análisis clásicos, algunos ejemplos clásicos son: T de Student: Compara medias entre dos grupos. Prueba T: Compara medias entre uno o dos grupos. ANOVA: Compara medias entre tres o más grupos. Regresión Lineal: Evalúa la relación entre variables. Correlación de Pearson: Mide la asociación lineal, por ejemplo, entre horas de estudio y desempeño en un examen. ¿Cuándo se utilizan? La variable es cuantitativa y continua. La población tiene distribución aproximadamente normal. No se conoce la desviación estándar poblacional. El tamaño muestral es pequeño (generalmente n < 30). Pruebas NO Paramétricas La estadística no paramétrica incluye métodos que no requieren que los datos se ajusten a modelos teóricos. Por ejemplo, distribuciones como la chi-cuadrada se determinan a partir de los propios datos. Algunos ejemplos comunes son: Wilcoxon: Compara dos grupos relacionados. Mann-Whitney U: Compara dos grupos independientes. Kruskal-Wallis: Compara más de dos grupos. Spearman: Evalúa correlación ordinal o no lineal. ¿Cuándo se utilizan? Cuando los datos no siguen una distribución normal. Cuando los datos son ordinales o categóricos. Cuando hay valores atípicos o distribuciones sesgadas. Cuando el tamaño de muestra es pequeño. Cuando se violan otros supuestos paramétricos.

📊 Pruebas Paramétricas

Las metodologías paramétricas suponen que los datos provienen de una población con distribución normal. Estas pruebas tienen mayor poder estadístico y son ampliamente utilizadas en análisis clásicos, especialmente cuando se trabaja con variables cuantitativas y continuas. Se aplican cuando la población tiene una distribución aproximadamente normal, no se conoce la desviación estándar poblacional y el tamaño muestral es pequeño (generalmente n < 30).

  • T de Student: Compara medias entre dos grupos.
  • Prueba T: Compara medias entre uno o dos grupos.
  • ANOVA: Compara medias entre tres o más grupos.
  • Regresión Lineal: Evalúa la relación entre variables.
  • Correlación de Pearson: Mide la asociación lineal, por ejemplo, entre horas de estudio y desempeño en un examen.

📈 Pruebas No Paramétricas

La estadística no paramétrica incluye métodos que no requieren que los datos se ajusten a modelos teóricos. Estas pruebas son útiles cuando los datos no siguen una distribución normal, son ordinales o categóricos, presentan valores atípicos o distribuciones sesgadas, o cuando se violan otros supuestos paramétricos. También se aplican en muestras pequeñas donde los métodos paramétricos no son apropiados.

  • Wilcoxon: Compara dos grupos relacionados.
  • Mann-Whitney U: Compara dos grupos independientes.
  • Kruskal-Wallis: Compara más de dos grupos.
  • Spearman: Evalúa correlación ordinal o no lineal.

Matrices de correlación

Una matriz de correlación, también conocida como tabla de calor, es una representación gráfica que muestra la relación entre diferentes variables en un conjunto de datos. Cada celda de la matriz representa el coeficiente de correlación entre dos variables, lo que permite identificar patrones y relaciones significativas de manera visual por medio de colores, donde los colores más fuertes representan correlación alta y los colores más suaves correlación baja, en algunos casos o rojo para correlación baja y azul para correlación alta facilitando la interpretación de los datos. Esto es útil debido a que cuando tenemos muchisimos datos es muy dificil visualizar la correlación uno por uno, mientras que una matriz de correlación permite ver todas las relaciones de un vistazo. En nuestro caso particular usamos tablas de calor para visualizar la correlación que se obtuvo en dos tablas cuyos datos se habían obtenido mediante la prueba de correlación de Pearson. Se realizaron dos tablas de calor, en una podemos ver que la correlación más alta está en un color fuerte y la correlación baja está en un color más suave, mientras que en la otra podemos evidenciar que lo único que nos importa es visualizar la correlación alta.

Limpieza de datos

Es natural para nosotros trabajar con miles de datos que son reportado la gran mayoría en formatos de excel, csv, .json. Sin embargo estos datos son muy propensos a tener errores pues son recolectados de diferentes fuentes, en diferentes momentos, y recolectado por personas, lo cual es natural que por la cantidad de datos haya errores por lo que es común encontrar datos incompletos, inconsistentes o duplicados. OpenRefine es una herramienta poderosa que nos permite limpiar y transformar estos datos de manera eficiente. Limpiar datos es necesario para asegurar la calidad y precisión de los análisis posteriores y para poder acceder a la información obtenida de manera más fácil. Aunque en python es relativamente sencillo limpiar data, también existe la opción de openRefine. Esta aplicación funciona de manera similar a una hoja de cálculo, pero con capacidades avanzadas de limpieza y transformación de datos. Es muy intuitiva y fácil de usar OpenRefine ofrece una interfaz intuitiva que facilita la identificación y corrección de errores comunes en los datos, como valores faltantes, formatos inconsistentes o duplicados. Además, permite realizar transformaciones complejas en los datos, como la agrupación, la filtración y la reestructuración, lo que la convierte en una herramienta versátil para el análisis de datos.
